Estrutura base:
- pasta cliente: contém o código-fonte do site estático, desenvolvido usando React.js. Ela inclui componentes, páginas e 
  outros recursos relacionados ao front-end.

- pasta servidor contém o código-fonte do servidor de back-end, desenvolvido usando Node.js. Ela inclui controladores, rotas, 
  modelos de dados e configurações do servidor.

- arquivo .gitignore lista arquivos e pastas que devem ser ignorados pelo sistema de controle de versão (Git).

- arquivo .env pode conter variáveis de ambiente, como informações confidenciais, chaves de API e configurações específicas 
  do ambiente.
  
- arquivo netlify.toml é usado para configurar opções específicas do Netlify, se você estiver usando o Netlify para hospedar 
  o site estático.
